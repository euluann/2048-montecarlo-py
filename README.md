## 🎯 About This Project
This repository contains a Python implementation of **2048**, including variations using the **Monte Carlo algorithm**:

- **2048.py** – Full recreation of the classic 2048 game, playable manually.  
- **2048mc.py** – 2048 integrated with a Monte Carlo algorithm that plays automatically.  
- **2048mc_dynamic.py** – Monte Carlo version with a dynamic number of simulation clones: starts small and increases as the score grows to match the game's difficulty.
